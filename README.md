**Dimensions** is a Paper plugin that introduces a unique gameplay experience to your Minecraft server. Players can choose between two dimensions: Skyblock and Stoneblock, each offering its own advantages and challenges.

# ❗ Disclaimer
**This is a spoiler! Don't read the discription if you want an complitly unspoiled experience!**

# ⬆️ Leveling
You must level up six categories:

  - Mining (Pickaxe)
  - Foraging (Axe)
  - Digging (Shovel)
  - Farming (Hoe)
  - Combat (Sword, Bow & Crossbow)
  - Season (One of the above, changes with each Season)

You start with stone tools; as you advance through the story, you can buy and upgrade to new tools.

# 🔄 Seasons
With each season, all of your stats reset. A single season lasts between four to six months. Each season will feature new puzzles, bosses, and a new item to master. At the beginning of each season, you have the option to switch your dimension. Please keep in mind that you will lose all of your items, and all blocks and items in chunks claimed only by you will be deleted. If you play with a friend, you both have to claim chunks to be safe.

# 🙋 Villagers
At the beginning of each season, only the Basalt Villager will be available. As each season focuses on different items and blocks, you will have to farm accordingly.

# ℹ️ Further Information
While primarily developed for the *Dimensions Server Network*, this plugin is also available for private servers. Please note that **the experience scales with your community**: the larger the player base, the more engaging the gameplay becomes.

**These are not all the features planned for the plugin\!** I want to provide a clear description of the plugin without spoiling too much. Please also keep in mind that features like **stat resets and season updates will not happen automatically**. If you are not playing on the official *Dimensions Server Network*, **these features might not be supported**, as the admins themselves must manage them\!

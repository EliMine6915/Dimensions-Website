Dimensions
---
**Dimensions** is a Paper plugin that introduces a unique gameplay experience to your Minecraft server. Players can choose between two dimensions: Skyblock and Stoneblock, each offering its own advantages and challenges.

While primarily developed for the _Dimensions Server Network_, this plugin is available for private servers. Please note that the experience scales with your community, the larger the player base, the more engaging the gameplay becomes.

As a solo developer, I put a lot of heart into this project, so if you enjoy the plugin and are in a position to do so, a small tip would be greatly appreciated to support future updates.
